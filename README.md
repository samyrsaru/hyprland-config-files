# Hyprland Config Files

My complete Arch Linux + Hyprland configuration backup.

## What's Included

- **dotfiles/** - Shell configurations (.zshrc, .bashrc, .shell_common, .gitconfig)
- **config/** - Application configs:
  - `hypr/` - Hyprland window manager
  - `waybar/` - Status bar
  - `kitty/` - Terminal emulator
  - `nvim/` - Neovim editor
  - `rofi/` - Application launcher
  - `mako/` - Notification daemon
  - `yazi/` - File manager
  - `lazygit/` - Git TUI
  - `uwsm/` - Wayland session manager
  - `gtk-*/` - GTK themes
- **ssh/** - SSH keys and config
- **gnupg/** - GPG keys
- **scripts/** - Custom shell scripts
- **wallpapers/** - Desktop wallpapers
- **local/share/** - Fonts, themes, icons
